Funcionais Não Funcionais Inversos

Requisitos Funcionais

  1. Apenas alunos e professores da escola podem acessar o sistema.
  2. O sistema deve registrar quais livros os alunos estão lendo.
  3. O sistema deve registrar o tempo que os alunos gastam lendo livros.
  4. O sistema deve permitir que os professores anexem atividades complementares sobre os livros aos alunos.
  5. O sistema deve mostrar recordaçoes de leitura
  6. O sistema deve ter um ranking de livros mais lidos pelos alunos.
  7. O sistema deve ter um espaço para discussão sobre os livros.
  8. O sistema deve garantir que os alunos vejam quais livros estão disponíveis
  9. O sistema deve garantir que os alunos possam fazer reservas com antecedência dos livros disponíveis
  10. O sistema deve ter um lembrete para devolução dos livros
  11. O sistema deve permitir que os professores recomendam leituras para cada turma
  12. O sistema deve permitir que os professores recomendam leituras para cada turma
  13. O sistema deve permitir que os professores tenham acesso a relatórios sobre quais matérias estão sendo mais utilizadas
  14. O sistema deve garantir que o diretor da escola um painel onde ele monitore o uso da biblioteca
  15. O sistema deve mostrar os livros mais lidos nesse painel

Requisitos Não Funcionais

  1. O sistema deve garantir que apenas alunos e professores da escola tenham acesso ao sistema
  2. Apenas administrador pode ter acesso aos relatorios.
  3. Apenas administrativo podem acomponhar o desempenho dos alunos.
  4. Apenas administradores podem ver quem mais acessa o sistema.
  5. O sistema deve disponibilizar alguns livros no formato digital
  6. Apenas professores podem anexar atividades complementares aos livros.
  7. Somente os professores podem fazer leitura recomendada para cada turma.
  8. O sistema deve apresentar recomendacoes de livros baseados no que o usuario leu.
  9. O sistema deve ter uma data limite para atraso da entrega de livro.

Requisitos Inversos

  1. O sistema não deve permitir que o aluno precise pedir indicações de livros
  2. Apenas administradores podem ver quem mais acessa o sistema.
  3. Somente os professores podem fazer leitura recomendada para cada turma.
  4. Pessoas externas não podem acessar o sistema.
  5. Alunos não podem ter acesso à frequência dos demais alunos.
  6. O sistema não deve aceitar livros devolvidos com atraso
  7. O sistema não deve aceitar que pessoas internas tenham acesso ao sistema
